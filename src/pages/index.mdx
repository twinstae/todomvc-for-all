이 레포지토리는 작은 실험장입니다.

서로 다른 프레임워크와, 심지어 서로 다른 언어로 같은 todolist를 구현합니다.

다음 링크를 클릭하시면, 해당 프레임워크로 구현된 예시를 보실 수 있습니다.

대부분은 프레임워크에 무관하게 공통된 방식을 사용합니다.

특정 언어나 기술에 종속되는 부분은 갈아끼우기 쉽게 만듭니다.

## 상태관리

- react: useState, useEffect, zustand, jotai
- vue: ref, reactive, watch, pinia
- svelte: let, store
- solid: createSignal, createEffect
- lit: property, state
- elm: model
- agnostic: redux-toolkit, nanostore

## 스타일

- sass
- css module
- tailwind, daisyui
- vanilla extract
- classless
